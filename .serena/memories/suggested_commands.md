# Suggested Commands
- Install dev tooling: `pip install -e ".[dev]"`.
- Install test-only tooling: `pip install -e ".[test]"`.
- Full local verification: `make verify` (runs `ruff check --fix .`, `ruff format .`, `ruff check .`, `pre-commit run --all-files`, then pytest).
- Format/fix: `make format`.
- Lint: `make lint`.
- Run pre-commit hooks: `make pre-commit`.
- Run all tests per Makefile: `make test` (runs `PYTHONPATH=. pytest tests/unified_ui/`, then `PYTHONPATH=. pytest tests/`, so unified UI tests run twice).
- Focused tests: `PYTHONPATH=. pytest tests/unit/api/test_quote.py`, `PYTHONPATH=. pytest tests/unified_ui/test_registry.py`, or `PYTHONPATH=. pytest path/to/test.py::test_name`.
- Whitespace check for docs-only edits if pre-commit is unavailable: `git diff --check -- <file>`.
- OpenSpec/OpenCode workflow exists under `.opencode/commands/opsx-*.md` and `openspec/`; use `/opsx-propose`, `/opsx-apply`, `/opsx-explore`, `/opsx-archive` only when requested.
